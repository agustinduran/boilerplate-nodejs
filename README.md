# Promiedos Fueguino

Proyecto realizado por Agustín Durán

## Tecnologías utilizadas

### Sprint-1
* JavaScript
* NodeJs
* Express, Helmet, Cors
* Postman
* NPM
* Swagger OpenApi
* MariaDB
* ORM Sequelize
* Principios SOLID.
* Arquitectura Hexagonal DDD
* JWT (Json Web Tokens)
* BCrypt para cifrado de contraseñas
* Metodología TDD. Pruebas Automatizadas con Jest y Supertest

## Instalación

```bash
npm i
npm run dev
```
